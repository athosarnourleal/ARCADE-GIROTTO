# ARCADE-GIROTTO
uma coleção de 3 minigames feita em C para um trabalho do progessor Pedro Girotto
## perguntas e respostas
5 perguntas cada uma com 4 alternativas
## cobra na caixa
um jogo de sorte onde deve-se escolher entre 5 caixas e agir de acordo com o conteudo no interior delas: se for um botão você ganha e se for uma cobra o outro ganha.
## gousmas war
um jogo de estrategia em que os jogadores devem administrar os pontos de furia e atacar para destruir as gousmas do inimigo e ganhar
